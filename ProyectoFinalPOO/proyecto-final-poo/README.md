## Descripción

Sistema de gestión de productos desarrollado en Java utilizando Programación Orientada a Objetos.

El sistema permite:

- Registrar productos.
- Buscar productos.
- Actualizar productos.
- Eliminar productos.
- Mostrar todos los productos.

La persistencia de datos se realiza mediante Hibernate (JPA) y MySQL.

## Tecnologías

- Java 17
- Maven
- Hibernate ORM
- JPA
- MySQL Workbench

## Conceptos aplicados

- Clases
- Objetos
- Encapsulamiento
- Interfaces
- Herencia
- Polimorfismo
- CRUD
- DAO
- Persistencia con Hibernate

## Cómo ejecutar

1. Crear la base de datos:

```sql
CREATE DATABASE tienda;
```

2. Configurar usuario y contraseña en `persistence.xml`.

3. Ejecutar `Principal.java`.

4. Seleccionar diferentes opciones para implementación de datos.
